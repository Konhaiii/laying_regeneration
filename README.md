# Lying Regeneration
https://modrinth.com/project/laying-regeneration

**Lying Regeneration** is a simple datapack that allows players to regenerate health while lying in a bed.

## Features

Players regenerate health over time when lying in a bed.\
**Regeneration only occurs if the night is not skipped.**

This can happen for example if :

- Not all players are sleeping on a multiplayer server.
- The players_sleeping_percentage gamerule is set above 100.
- Another mod prevents skipping the night, such as BetterDays.

**Players do not receive any regeneration if the night is successfully skipped.**

## Notes

This datapack works in both singleplayer and multiplayer environments. It is particularly suited for setups where sleeping does not skip the night or requires coordination between players.

## Installation

Place the datapack in the world's datapacks folder and reload or restart the world.
